#  Netflix Data Analysis

##  About the Project

In this project, I explored a Netflix dataset to understand patterns in movies and TV shows.
The main goal was to analyze how content is distributed across different categories like type, country, year, rating, and duration.

---

##  What I Tried to Find

* How many Movies vs TV Shows are available
* Which countries produce the most content
* How Netflix content has grown over the years
* Distribution of content ratings
* Duration trends of movies
* Popular genres on Netflix

---

##  Analysis Performed

I performed the following analysis using Python:

1. Movies vs TV Shows Distribution
2. Top Content Producing Countries
3. Content Added Over the Years
4. Content Rating Distribution
5. Movie Duration Distribution
6. Genre Analysis

All graphs and visualizations are included in the notebook.

---

##  Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook (VS Code)

---

## 📂 Project Files

* `Netflix_Data_Analysis.ipynb` → contains complete code and visualizations
* `netflix_dataset.csv` → dataset used for analysis

---

##  How to Run

1. Download or clone the repository
2. Open the notebook in VS Code or Jupyter Notebook
3. Run all cells to view the analysis and graphs

---

##  Key Observations

* Movies are more in number compared to TV Shows
* Content growth increased rapidly after 2015
* A few countries contribute most of the content
* Most content falls under specific rating categories
